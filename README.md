# CITL LLMOps Studio v1.5

> **Shared Authorship** — CITL Development Team, Renton Technical College
> **Project Lead:** Doc McDowell

A complete local LLM operations environment for IT education. GUI for managing Ollama models, writing Modelfiles, building custom training databases, and running translation/transcription workflows — all offline.

## What It Does

- **Model Manager** — pull, list, delete, and switch Ollama models from a GUI
- **Modelfile Editor** — write and test custom SYSTEM prompts and model parameters
- **Database LLMOps Builder** — structure knowledge bases for fine-tuning and RAG pipelines
- **Translation Studio** — offline multilingual document translation
- **Presentation Suite** — generate and deliver AI-powered slide decks using local LLMs
